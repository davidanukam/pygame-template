# Pygame Project Template

A boilerplate for starting Pygame development quickly with modern Windows styling.

---

## Features
- **Clean Start:** Silences the Pygame welcome message on import.
- **Dark Mode:** Custom Windows header color via `pywinstyles`.
- **Pre-configured:** 1280x720 resolution with 60 FPS clock sync.

---

## Setup Instructions

1. **Create a Virtual Environment (Optional but recommended):**
```bash
   python -m venv venv
   source venv/Scripts/activate  # Windows
```
2. **Install the required libraries**

```bash
   pip install -r requirements.txt
```
3. **Run the project**
```bash
   python main.py
```
